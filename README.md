This site is hosted at <https://cs107e.github.io/>.

For local preview:

- Install tools on local system
   - ruby, bundler, jekyll 
   - see instructions here <https://jekyllrb.com/docs/installation/>
- Start server
    + `bundle exec jekyll serve`
- View in browser at url <http://localhost:4000>
    + live reload enabled
